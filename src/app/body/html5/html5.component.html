2) What is difference between LocalStorage and cookies?
Local storage: Cookies:
1)Local storage stores the data in the form of key and value. 1)Good for small amount of data.
2)The data is stored for longer duration and remains in the browser unless user delete it explicitly, it remains even after computer restart. 2)Difficult to work with JavaScript.
3)No way to specify the time out period as the Cookies have. 3)All data is transferred to and from server, so bandwidth is consumed on every request.
4)Good to store large amount of data, up to 4MB. 4)Can specify timeout period so that cookies data are removed from browser.
5)Easy to work with the JavaScript.
6)Local storage data is not sent to the server on every request (HTTP header) as it is purely at the client side.

4) What is difference between svg and canvas?
SVG Canvas
1)SVG has better scalability. So it can be printed with high quality at any resolution. 1)Canvas has poor scalability. Hence it is not suitable for printing on higher resolution.
2)SVG gives better performance with smaller number of objects or larger surface. 2)Canvas gives better performance with smaller surface or larger number of objects.
3)SVG can be modified through script and CSS. 3)Canvas can be modified through script only.
4)SVG is vector based and composed of shapes. 4)Canvas is raster based and composed of pixel.

7) What Is Geoloaction in HTML5? Give me example?

The HTML5 geolocation feature lets you find out the geographic coordinates (latitude and longitude numbers) of the current location of your website's visitor. This feature is helpful for providing better browsing experience to the site visitor. For example, you can return the search results that are physically close to the user's location.
